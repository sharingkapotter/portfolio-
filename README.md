# Sunil Sodhi — Portfolio

Personal portfolio site built with React + Vite + Tailwind CSS.

## Stack
- React 18 + Vite
- Tailwind CSS v3
- React Router v6
- Lucide React icons
- Fonts: Syne, DM Sans, JetBrains Mono (Google Fonts)

## Local Dev
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Vercel auto-detects Vite — just click Deploy
4. Your site will be live at `your-project.vercel.app`

## Update Content
All site content lives in one file: `src/data/profile.js`
- Update your summary, experience, skills, projects, certs there
- No need to touch any page components for content changes
